# How To Run This Project

### For first time only !
- `git clone https://github.com/umararrosyad/sistem-gudang`
- `cd sistem-gudang`
- `composer install`
- `cp .env.example .env`
- sesuaikan bagian database degan konfigurasi masing masing
- `php artisan key:generate`
- Buat database dengan nama `sistem_gudang`
- `php artisan migrate`
- `php artisan db:seed`
- `php artisan serve`
